## Slide 6 — Partnership Model

### On-Slide Copy

**How we work with ecosystem partners.**

- **Co-build** — Joint protocol development with shared IP ownership. Ideal for chains and foundations.
- **Integration** — HELIOS agent deployment on partner protocols. Plug-in model with licensing terms.
- **Infrastructure** — White-label protocol components for partner ecosystems. Design + deploy + maintain.

We don't compete with our partners. We build the layer underneath.

### Speaker Notes

- Three models, each for a different type of partner.
- Co-build is for deep relationships (L1/L2 chains, foundations).
- Integration is for existing protocols that want agent capabilities.
- Infrastructure is for ecosystems that want to offer DeFi primitives to their users.
- Tailor this slide to the specific partner in the room.
